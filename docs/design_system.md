# Sistema de Diseño - ORGI

## Paleta de Colores

```css
:root {
  --primary: #1A56DB;
  --secondary: #16BDCA;
  --success: #0EA472;
  --danger: #E02424;
  --warning: #FF8800;
  --neutral-900: #111928;
  --neutral-600: #6B7280;
  --neutral-200: #E5E7EB;
  --neutral-50: #F9FAFB;
  --background: #FFFFFF;
}
```

| Token | Color | Hex | Uso |
|-------|-------|-----|-----|
| Primary | Azul | #1A56DB | Botones, links, header |
| Secondary | Teal | #16BDCA | Énfasis, badges |
| Success | Verde | #0EA472 | Ingresos, positivo |
| Danger | Rojo | #E02424 | Gastos, negativo, errores |
| Warning | Naranja | #FF8800 | Deudas, alertas |
| Neutral 900 | — | #111928 | Texto principal |
| Neutral 600 | — | #6B7280 | Texto secundario |
| Neutral 200 | — | #E5E7EB | Bordes, separadores |
| Neutral 50 | — | #F9FAFB | Fondos de cards |
| Background | — | #FFFFFF | Fondo principal |

## Tipografía

- **Display**: Inter (Google Fonts) — títulos y dashboard
- **Body**: Inter — texto general
- **Monospace**: JetBrains Mono — valores monetarios

Escala: 12px / 14px / 16px / 20px / 24px / 32px / 48px
